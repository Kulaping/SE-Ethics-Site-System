# SE Ethics

This is my first full-stack web application (project name subject to change).  
This project is currently under active development.

The platform allows users to **view, read, and comment on articles**, featuring:

- Live news updates powered by the NewsAPI.org API  
- Simple weather tracking using the OpenWeatherMap API  
- Dynamic comment system *(work in progress)*

---

## Project Status

This project is still being built. Some features, including comment system enhancements and caching improvements, are under development.

---

## Features

- User authentication using **JSON Web Tokens (JWT)**
- Secure password hashing via **Werkzeug Security**
- Custom-built **caching mechanism** to reduce excessive API calls
- Planned migration to **Redis** for improved caching performance
- RESTful API architecture for end-to-end communication
- Dynamic article interaction system
